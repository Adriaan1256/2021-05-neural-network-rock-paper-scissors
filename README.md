# Neural Network Rock-Paper-Scissors Predictor

Course: EAI320 – Intelligent Systems  
University of Pretoria  
Completed: May 2021

---

## Project Overview

This project involved implementing a neural network capable of learning patterns in gameplay data to predict the next move in a Rock-Paper-Scissors match.

The neural network was trained using datasets provided in CSV format. After training, the model was used to compete against other automated players by predicting the most likely next move based on previous sequences of moves.

The project also analysed the effect of different neural network hyperparameters such as learning rate, batch size, epochs, and network architecture.

## Objectives

- Implement a neural network from scratch
- Train the model using gameplay data
- Predict optimal moves in a Rock-Paper-Scissors game
- Evaluate neural network performance
- Analyse the impact of hyperparameters including:
  - Learning rate
  - Batch size
  - Epoch count
  - Number of layers
  - Number of neurons per layer
